# Cozy Pond 🎣

Et koselig, prosedyre-tegnet fiskespill som kjører rett i nettleseren – ingen bilder, alt tegnes på canvas.

## Spill det
Åpne `index.html` lokalt, eller spill live via GitHub Pages.

## Funksjoner
- 6 ulike vann (Skogstjernet, Trollmyra, Stryket, Elgtjernet, Fjellvatnet, Nordlysvatnet) med egen stemning, dyreliv og sjeldne troféfisk
- 10 fiskearter + legendariske rariteter
- 6 fiskestenger med stigende egenskaper
- Marked med fiskehandler, kiosk, kasino og lisenskontor
- Prosedyre-generert lyd via Web Audio + mp3-samples
- Intro-cinematic, nordlys, elg, skytende stjerner og mer

## Teknisk
Ren HTML/CSS/JavaScript. Ingen byggesteg, ingen avhengigheter.

## Kjør lokalt
```
Åpne index.html i en nettleser.
```
